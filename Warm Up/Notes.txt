#1 If you add number to a string, the number gets converted to string and gets concatenated.
        let firstName = "Akshay";
        let lastName = 7;

        const fullName = firstName + lastName;
        console.log(fullName);

#2 If I add 'String' in between other 2 variables will also be treated as String too.
        let firstName = 3;
        let lastName = 7;

        const fullName = firstName + "" + lastName;
        console.log(fullName);

        let firstName = 3;
        let lastName = 7;

        const fullName = firstName + lastName;
        console.log(fullName);

#4 Key rule

        Primitive values (number, string, boolean) → passed by value

        Objects & arrays → passed by reference (technically “reference value”)

#5              Function	        What it does	                Example
                Math.round()	        nearest integer	                4.6 → 5
                Math.floor()	        always round down	        4.9 → 4
                Math.ceil()	        always round up	                4.1 → 5
                Math.abs()	        remove negative sign	        -8 → 8

        Rule for Math.round():
        decimal ≥ 0.5 → round up
        decimal < 0.5 → round down

#6 Function scope uses sligthly more memory than Block scope. 
        So, Declare variables as close as possible to where they are used.
